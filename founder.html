动态漏洞挖掘工具（LyScript）

V1.1.0
版权登记号：2023SR0475117, 2024SR1010782, 2026SR0014808

LyScript 是深度集成于 x64dbg 调试器的 Python 自动化测试框架，通过标准化 API 接口与 x64dbg 调试引擎无缝交互。该框架依托 Python 脚本化优势，为逆向工程、漏洞利用开发及恶意代码分析提供模块化解决方案，核心能力涵盖内存断点自动化漏洞挖掘、寄存器级精准调试控制、二进制指令流动态追踪及结构化反编译分析。用户可通过简洁的 Python 语法快速构建 POC 验证、模糊测试、ROP 链构造等高阶调试场景，结合跨平台远程调试协议，有效提升二进制安全研究中动态分析效率与漏洞复现精准度。此外，结合其配套开发的 x64dbgMCP 工具集并对接 Cherry Studio 工具，可驱动 x64dbg 实现智能化逆向分析、漏洞挖掘与处理，全面提升逆向分析效率，实现调试操作的自动化。[5]

静态漏洞挖掘工具（IDA Moles）

V1.0.7
版权登记号：2026SR0130174

IDA Moles 是深度集成于 IDA Pro 的专业静态逆向分析接口工具，通过标准化的调用逻辑与 IDA Pro 逆向引擎实现无缝交互，依托 Python 生态的简洁编程优势，为二进制逆向分析、恶意代码溯源、漏洞挖掘辅助等场景提供全流程的程序化解决方案，核心能力涵盖高效反编译控制、高级调试操作、深度内存分析、智能函数解析、MCP 服务器扩展及自动化批量处理，全方位满足逆向工程中的各类复杂分析需求。[53]

PE 分析工具（PeView）

V3.1.12
版权登记号：2023SR0490503

PEView 是一款由 C/C++ 开发的轻量化、AI 驱动的 Windows PE/PE32+ 文件专业解析工具，主打静态解析、AI 赋能、多版本适配，专为逆向分析、反病毒检测及病毒木马样本深度解包分析设计，是安全研究人员的核心生产力工具。工具支持 EXE、DLL 等多格式及 32/64 位 PE 文件解析，可零执行风险下完成 DOS 头、NT 头等全维度原子级结构解析与查询，还能对接各类大模型实现对话式智能交互，快速生成结构化分析报告；内置 12+ 核心工具，支持解析结果可视化、多格式导出，以及交互式修改文件属性、构造 POC 样本等操作。工具适配 Windows 全系统，提供免费独立版、专业接口版、企业级 MCP 旗舰版三个版本，按需满足基础解析、开发集成、AI 旗舰分析等场景需求，支持命令行、服务端、Python 接口多方式操作，无恶意捆绑且支持商业定制开发，大幅提升逆向工程、数字取证等场景的分析效率。[7]

Windows 动态反汇编调试软件（LyDebugger）

V2.0.0 / V2.1.0
版权登记号：2023SR0507262, 2026SR0130049

LyDebug（LyDebugger）是一款由 C/C++ 自主开发的轻量级 CLI 版 Windows 应用层动态调试器，无第三方库依赖，是逆向分析、程序排错、漏洞研究领域的实用工具。该工具依托 Windows 系统调试 API 与 Capstone 反汇编引擎，完美支持 x86/x64 双架构程序调试，以命令行交互为核心模式，提供原子级的调试操作能力，作为轻量化调试工具，LyDebug 具备运行高效、操作直接的特点，无需复杂配置即可使用，能为逆向工程师、安全研究人员提供精准、稳定的动态调试支持，目前已形成完整的调试器雏形，适配恶意代码分析、漏洞挖掘、程序调试等各类技术场景。[11][52]

内核级内存读写驱动软件（LyMemory）

V2.0.0
版权登记号：2023SR0490504

LyMemory 是一款基于 Windows 内核态内存访问协议（KMAP）构建的模块化驱动级工具套件，其核心能力涵盖跨进程内存读写、内核对象操作及硬件虚拟化层监控。该工具通过创新性的分页表遍历算法实现物理内存级精确访问，支持多级指针链解析、内存池特征检索、分页属性动态探测等逆向工程关键场景，提供包括 EPT Hook 拦截、SMEP/SMAP 绕过、内存保护属性篡改等内核攻防对抗功能模块。用户可通过 Python 扩展接口构建自动化内存分析流水线，配合 SSDT/HAL Dispatch Table 监控模块，显著提升漏洞利用构造、反作弊对抗及 Rootkit 检测等场景的操作效率，驱动需在 Windows 调试模式或经过 WHQL 认证的数字签名环境下部署以实现完整功能矩阵。[6]

反汇编代码注入软件（LyInjector）

V2.0.0
版权登记号：2023SR0490502

LyInjector 是一款由 C/C++ 精心打造的专业应用层汇编代码注入工具，主要面向渗透测试工程师群体。它具备强大且实用的功能，用户能够借助它向指定的应用层进程强行注入 DLL 模块或者 ShellCode 指令集。同时该工具还支持对第三方进程进行汇编级的 Call 调用，也可用于特定 ShellCode 的完整性测试，确保其在目标环境中能正常运行。通过使用这款工具，极大地拓展了操作的灵活性。[8]

命令行版远程控制软件（LySocket）

V1.3.0
版权登记号：2023SR0507263

LySocket 是一款基于 C/S 架构的专业级远程管理工具，支持可自定义的套接字通信协议（如 TCP/UDP）及多用户协同管理模式。该工具深度融合系统指令执行、跨网络文件传输、硬件资源监控（CPU/内存/磁盘）等核心功能，结合 TLS 加密通信协议保障数据传输安全。通过 ShellCode 注入、内存取证与进程注入等底层操作，适用于渗透测试、权限维持、漏洞验证等网络安全场景。其多用户权限维持机制支持网络波动环境下的稳定会话控制，适用于企业级安全攻防演练与后渗透远程运维管理。[9]

内网穿透流量转发软件（TcpForward）

V1.0.0
版权登记号：2023SR0507815

TcpForward 是一款高性能 TCP 流量代理工具，专为渗透测试与内网渗透场景设计，基于 Socket 多路复用及流量定向路由技术，实现跨网络域安全隧道构建。核心功能涵盖端口映射、流量重定向、内外网穿透，支持动态规则配置与多级跳板中转，可精准转发特定主机与端口的流量路径，突破网络隔离策略，有效辅助红队攻击链中的横向移动、隐蔽通信及漏洞利用环节，提升攻防演练效率。[10]

外部 D3D 动态绘制库软件（D3DDrawLib）

V1.0.0
